Welcome to Faculty Audit System

The Faculty Audit System is a complicated process and requires time and effort.Until now ,this process was carried out Manually. This project is about to introduce the environment for faculties to handle their work and maintain records easily instead of making a lot of paperwork. An effective and efficient way of managing all the stuff can be done easily. Better secure transfer of all information provided using the latest technologies. Updating, Inserting, and all the operations can be done easily and efficiently. This application will be most useful in the future so that reducing paperwork will affect the time as well as the environment.
